# 📋 Taskati - Task Manager App

Taskati is a simple and modern **Task Management App** built using **Flutter**.  
The app helps users organize their daily tasks, track progress, and stay productive.

---

## 🚀 Features

- 👤 User Profile Setup (Name + Image)
- 📅 Daily Task Progress
- 📆 Date Picker for Tasks
- 📊 Task Status Tracking
- 🟣 Tabs for Task Filtering
  - All Tasks
  - In Progress
  - Completed
- 💾 Local Data Storage using SharedPreferences

---

## 📱 Screens

### Splash Screen
Displays app logo and checks if user profile exists.

### Upload Profile
User uploads:
- Profile image
- Name

### Home Screen
Shows:
- Profile header
- Daily progress
- Task list
- Task filtering tabs
